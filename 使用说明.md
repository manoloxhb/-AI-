直接双击打开 `.html` 文件（即使用 `file://` 协议）通常**无法正常运行**，原因如下：

1. **摄像头权限限制**：浏览器通常禁止 `file://` 协议访问摄像头。
2. **CORS 跨域限制**：MediaPipe 和 Three.js 需要加载外部资源（WASM 文件等），本地直接打开会被浏览器安全策略拦截。

你需要启动一个**本地服务器 (Local Server)** 来运行它。以下是几种最简单的方法：



### 方法一：使用 VS Code (最推荐)



如果你使用 Visual Studio Code 编辑代码：

1. 安装插件：在左侧扩展商店搜索并安装 **Live Server**。
2. 右键运行：在 `index.html` 编辑区点击右键，选择 **"Open with Live Server"**。
3. 结果：浏览器会自动弹出，地址通常是 `http://127.0.0.1:5500/index.html`，此时摄像头和手势识别均可正常工作。



### 方法二：使用 Python (无需安装额外软件)



如果你的电脑装有 Python（Mac 系统通常自带）：

1. 打开终端 (Terminal 或 CMD)。

2. `cd` 进入你的 `index.html` 所在的文件夹。

3. 输入以下命令并回车：

   ```
   # 如果是 Python 3
   python -m http.server 8000
   
   # 如果是 Python 2
   python -m SimpleHTTPServer 8000
   ```

4. 打开浏览器访问：`http://localhost:8000`
